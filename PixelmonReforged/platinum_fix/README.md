
## plutinum_fix

v9.2系のプラチナ鉱石の出現設定の一部がルビー鉱石になっているバグを修正するデータパックです。

以下の設定のうち、下2つがルビー鉱石となっているのを修正します。

- プラチナ鉱石の出現場所：1チャンクのY-16～-144までに5個の小さな鉱脈
- プラチナ鉱石の出現場所：1チャンクのY-16～-144までに4個の通常サイズの鉱脈
- プラチナ鉱石の出現場所：1チャンクのY80～-80までに1/9個の大きな鉱脈
