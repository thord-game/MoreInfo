
## spawning-fix

v9.2.8 のタイプ：ヌルの出現設定の誤りにより出現しないバグを修正するデータパックです。

タイプ：ヌルはエンド要塞で出現する設定となっていますが、エンド要塞かつ地上で出現する設定となっており、
これをエンド要塞かつ室内に変更することで出現させるように修正するデータパックです。
